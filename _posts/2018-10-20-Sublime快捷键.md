## 快捷键
> 全局搜索: cmd + p  
```
 输入文件名直接打开,支持单词首字母匹配。@后跟符号，#后跟文件名,:后跟指定行。  
 eg: tp@rf， 跳转到 text_parser.py 中的 read_file 方法。  
```
> 命令输入框： shift + cmd + P  
> 关闭当前tab: cmd + w

### 编辑
> 加选同名变量：cmd + D  
> 多行选中编辑： shift + cmd + L   
> 拷贝当前行： shift + cmd + d  
> 删除当前行： shift + ctrl + k  
> 移动行位置： ctrl + cmd + 上/下  

## 鼠标 + 快捷键

> 跳转到定义： 鼠标点选 + F12
